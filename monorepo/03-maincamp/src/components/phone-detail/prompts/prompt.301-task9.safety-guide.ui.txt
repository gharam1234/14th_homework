아래의 조건을 모두 적용하여, 아래의 요구사항을 모두 구현할 것.
구현 결과를 체크리스트로 반환할 것.

==============================================

조건-커서룰) 아래의 커서룰을 적용하여 작업하고, 이 작업이 끝나면 해당 rules 적용 결과를 체크리스트로 반환할 것.
            - @01-common.mdc
            - @04-func.mdc

조건-파일경로) 구현될 컴포넌트 파일: src/components/phone-detail/index.tsx (guide 영역)

==============================================

핵심요구사항) 안전거래 가이드 섹션을 구현할 것. 다음의 요소들을 step-by-step으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

            1) 직거래 팁 섹션
                - 제목: "직거래 팁"
                - 팁 항목 (최소 3개 이상):
                    * "밝은 장소에서 거래하세요"
                    * "카메라나 정품 인증서를 확인하세요"
                    * "거래 내역을 기록해두세요"
                    * "신분증 확인 후 거래하세요" (선택)
                - 각 팁마다 아이콘 표시
                - 리스트 형식으로 표시

            2) 사기 예방 정보 섹션
                - 제목: "이런 거래는 피하세요"
                - 사기 유형 (최소 3개 이상):
                    * "선입금을 요구하는 경우"
                    * "가격이 시세보다 현저히 낮은 경우"
                    * "서로 다른 플랫폼에서의 거래"
                    * "정품 인증서 없는 경우" (선택)
                - 각 항목마다 경고 아이콘 표시
                - 리스트 형식으로 표시

            3) 환불 정책 섹션
                - 제목: "환불 정책"
                - 정책 내용:
                    * "구매 후 7일 이내 환불 가능"
                    * "반드시 초기 상태여야 함"
                    * "배송료는 구매자 부담"
                    * "손상된 경우 환불 불가"
                - 각 정책을 명확하게 표시
                - 리스트 형식으로 표시

            4) 추가 정보 연결
                - 각 섹션 하단에 "자세히 보기" 또는 "더 알아보기" 링크 (선택)
                - 외부 링크 또는 모달로 상세 정보 표시

            5) 시각적 계층화
                - 섹션별 배경색으로 구분
                - 섹션 제목: 큰 글씨, 볼드 처리
                - 아이콘: 각 항목 좌측에 배치
                - 간격: 각 항목 간 충분한 여백

            6) 반응형 디자인
                - 데스크탑: 3열 레이아웃 (직거래 팁 | 사기 예방 | 환불 정책)
                - 태블릿: 2열 레이아웃
                - 모바일: 1열 레이아웃 (세로 정렬)

            7) 인터랙션
                - 섹션 제목 클릭 시 내용 확장/축소 (아코디언)
                - 스무스 애니메이션 적용
                - 호버 효과: 항목 배경색 변화

            8) 테스트
                - 모든 섹션이 올바르게 렌더링되는지 확인
                - 아이콘이 각 항목 옆에 표시되는지 확인
                - 텍스트 내용이 명확하게 보이는지 확인
                - 반응형 레이아웃이 올바르게 적용되는지 확인
                - 아코디언 기능이 작동하는지 확인 (선택)
