핵심요구사항) 이미지 갤러리 UI를 가로 스크롤 방식으로 구현할 것.

1) 이미지 갤러리 레이아웃
    - 가로 스크롤 가능한 이미지 리스트
    - 이미지 간격: 적절한 gap 적용
    - 스크롤바 스타일: 깔끔한 디자인 적용 (또는 숨김)
    - 이미지 크기: 일정한 크기로 썸네일 표시

2) 5장 이상 이미지 처리
    - 이미지가 5장 이상인 경우, 5번째 이미지 위치에 "+N" 오버레이 표시
    - "+N" 표시: 나머지 이미지 개수 표시 (예: 총 8장이면 "+3" 표시)
    - 오버레이 스타일: 반투명 배경 + 중앙 정렬된 텍스트
    - 5장 미만인 경우: 모든 이미지를 그대로 표시

3) 반응형 처리
    - 터치 디바이스에서 스크롤 가능하도록 설정
    - overflow-x: auto 적용
    - 모바일/데스크탑 모두 고려

4) data-testid 적용
    - 갤러리 컨테이너: data-testid="image-gallery"
    - 각 이미지: data-testid="gallery-image-{index}"
    - "+N" 오버레이: data-testid="gallery-more-count"

핵심요구사항) 이미지 데이터는 props로 전달받을 것.

1) Props 인터페이스
    - images: string[] (이미지 URL 배열)

2) 이미지 렌더링
    - 배열을 순회하며 이미지 표시
    - 각 이미지에 alt 속성 적용 (예: "중고폰 이미지 {index}")

핵심요구사항) CSS는 styles.module.css를 사용하여 구현할 것.

1) 클래스 네이밍
    - 갤러리 컨테이너: .gallery
    - 이미지 래퍼: .imageWrapper
    - 이미지: .image
    - 오버레이: .moreOverlay
    - 오버레이 텍스트: .moreCount

2) 스타일 요구사항
    - flexbox 활용
    - gap을 사용한 간격 조정
    - object-fit: cover로 이미지 비율 유지
    - transition 효과 적용 (hover 시)
