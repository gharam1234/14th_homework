구현 결과를 체크리스트로 반환할 것.

조건-커서룰) 아래의 커서룰을 적용하여 작업하고, 이 작업이 끝나면 해당 rules 적용 결과를 체크리스트로 반환할 것.
            - @01-common.mdc
            - @02-wireframe.mdc

조건-파일경로) 구현될 컴포넌트 폴더: src/components/phone-detail/
조건-파일경로) 구현될 TSX 파일경로: src/components/phone-detail/index.tsx
조건-파일경로) 구현될 CSS 파일경로: src/components/phone-detail/styles.module.css
조건-파일경로) 연결될 PAGE 경로: src/app/(protected)/phones/[id]/page.tsx

==============================================

핵심요구사항) 폰 상세 정보 페이지의 완성된 와이어프레임 컴포넌트를 페이지에서 import하여 연결시킬 것.
                - 연결될 경로: src/app/(protected)/phones/[id]/page.tsx

핵심요구사항) HTML과 flexbox를 활용한 와이어프레임 구조만 만들어낼 것. (스타일, 데이터바인딩은 나중 단계)

핵심요구사항) 다음의 10가지 영역을 모두 포함하여 구현할 것.
                1) 폰 이미지 갤러리
                    - imageGallery: 1280 * 420
                    - 메인 이미지 표시 영역 + 썸네일 슬라이더

                2) 폰 상세 정보
                    - basicInfo: 1280 * 140
                    - 모델명, 상태, 가격(할인가/원가), 해시태그, 액션버튼(신고/공유/위치/북마크)

                3) 폰 상세 스펙 섹션
                    - specs: 1280 * 280
                    - 제조사, 모델, 출시년도, RAM, Storage, 배터리 건강도, 색상, 단말기 상태

                4) 거래 위치 지도
                    - map: 1280 * 320
                    - 거래 장소 표시, 위도/경도 기반, 거래 가능 여부

                5) 구매 문의 기능
                    - inquiry: 1280 * 180
                    - 문의 개수 선택, 문의 제출 버튼

                6) 문의/답변 목록
                    - inquiryList: 1280 * 420
                    - 구매자 문의, 판매자 답변, 구매자 추가 문의, 문의 삭제/수정

                7) 판매자 정보 카드
                    - seller: 1280 * 160
                    - 판매자 프로필(프사/닉네임), 거래평점, 온도(신뢰도), 판매 물품 개수, 팔로우 버튼

                8) 가격 및 거래 정보
                    - priceInfo: 1280 * 120
                    - 판매가격, 정가 대비 할인율, 거래 방식 안내, 거래 안전 정보

                9) 안전거래 가이드
                    - guide: 1280 * 220
                    - 직거래 팁 섹션, 사기 예방 정보, 환불 정책

                10) gap (여백)
                    - gap: 1280 * 40 (각 영역 사이의 여백)

핵심요구사항) 각 영역의 구조를 다음과 같이 구성할 것.
                1) imageGallery:
                    - 메인 이미지 컨테이너 (1280 * 350)
                    - 썸네일 이미지 슬라이더 (1280 * 70)

                2) basicInfo:
                    - 모델명/상태 (왼쪽) + 가격 (오른쪽)
                    - 해시태그 영역
                    - 액션버튼 영역 (신고, 공유, 위치, 북마크)

                3) specs:
                    - 제조사, 모델, 출시년도 (3열)
                    - RAM, Storage, 배터리 건강도 (3열)
                    - 색상, 단말기 상태 (2열)

                4) map:
                    - 지도 임베드 영역 (1280 * 280)
                    - 거래 가능 여부 표시 (하단)

                5) inquiry:
                    - 문의 개수 선택 필드
                    - 문의 제출 버튼

                6) inquiryList:
                    - 문의별 구분선
                    - 각 문의 항목 (구매자 이름, 문의 내용, 답변 상태)
                    - 답변 컨테이너
                    - 문의 없을 때 메시지

                7) seller:
                    - 판매자 프로필 영역 (왼쪽) - 프사 + 닉네임
                    - 판매자 정보 영역 (중앙) - 평점, 온도, 판매 물품
                    - 팔로우 버튼 (오른쪽)

                8) priceInfo:
                    - 판매가격 (왼쪽)
                    - 할인율 (중앙)
                    - 거래 방식/안전 정보 (오른쪽)

                9) guide:
                    - 직거래 팁 (1개 섹션)
                    - 사기 예방 정보 (1개 섹션)
                    - 환불 정책 (1개 섹션)

핵심요구사항) 각 영역은 아래의 수치값을 그대로 반영할 것. (단위: px)
                - imageGallery: 1280 * 420
                - gap: 1280 * 40
                - basicInfo: 1280 * 140
                - gap: 1280 * 40
                - specs: 1280 * 280
                - gap: 1280 * 40
                - map: 1280 * 320
                - gap: 1280 * 40
                - inquiry: 1280 * 180
                - gap: 1280 * 40
                - inquiryList: 1280 * 420
                - gap: 1280 * 40
                - seller: 1280 * 160
                - gap: 1280 * 40
                - priceInfo: 1280 * 120
                - gap: 1280 * 40
                - guide: 1280 * 220
