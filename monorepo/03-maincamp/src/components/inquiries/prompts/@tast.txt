1. 문의 작성 기능@
문의 내용 입력 textarea (최대 100자)
실시간 글자 수 카운팅 (0/100 형태)
"문의 하기" 버튼으로 문의 등록
입력 validation

2. 문의 목록 조회@
등록된 문의 리스트 표시
각 문의별 정보:
작성자 프로필 이미지
작성자 닉네임
문의 내용
작성 날짜 (YYYY.MM.DD 형식)

3. 답변 작성 기능@
"답변 하기" 버튼 클릭 시 답변 입력창 표시
답변 내용 입력 textarea (최대 100자)
실시간 글자 수 카운팅
"취소" 버튼: 답변 작성 취소
"답변 하기" 버튼: 답변 등록

4. 답변 표시 기능@
문의에 대한 답변 트리 구조로 표시
답변 인덴트 처리 (왼쪽 24px 여백)
답변 아이콘 (return 아이콘) 표시
판매자 답변 표시:
판매자 프로필 이미지
"판매자" 레이블
답변 내용
작성 날짜
중첩 답변 지원 (답변에 대한 답변)

5. 답변 수정/삭제 기능@
수정 버튼 (연필 아이콘)
삭제 버튼 (X 아이콘)
판매자 답변에만 수정/삭제 버튼 표시

7. 권한 관리@
일반 유저: 문의 작성만 가능
판매자: 답변 작성, 수정, 삭제 가능
본인 작성 문의만 확인 가능 (필요 시)

8. 데이터 관리
문의 생성 API 연동
문의 목록 조회 API 연동
답변 생성 API 연동
답변 수정 API 연동
답변 삭제 API 연동
실시간 데이터 갱신
