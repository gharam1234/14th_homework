아래의 조건을 모두 적용하여, 아래의 요구사항을 모두 구현할 것.
구현 결과를 체크리스트로 반환할 것.

==============================================

조건-커서룰) 아래의 커서룰을 적용하여 작업하고, 이 작업이 끝나면 해당 rules 적용 결과를 체크리스트로 반환할 것.
            - @01-common.mdc
            - @02-wireframe.mdc
            - @03-ui.mdc

조건-파일경로) 구현될 컴포넌트 경로: src/components/inquiries/index.tsx

==============================================

핵심요구사항) 기존 답변(reply) 영역에 수정/삭제 버튼 UI를 추가할 것.
                1) 버튼 배치
                    - 위치: 답변 영역 우측 상단
                    - 방향: 가로 배치 (gap: 8px)
                    - 숨김 여부: 나중에 조건부 렌더링할 예정이므로, 현재는 항상 표시

                2) 수정 버튼 (연필 아이콘)
                    - 아이콘: 연필 모양 (svg 또는 icon 라이브러리 활용)
                    - 크기: 16px × 16px
                    - 배경: 투명 (hover 시 배경색 추가 가능)
                    - onClick: 아직 구현하지 않고, 스텁 상태 유지

                3) 삭제 버튼 (X 아이콘)
                    - 아이콘: X 모양 (svg 또는 icon 라이브러리 활용)
                    - 크기: 16px × 16px
                    - 배경: 투명 (hover 시 배경색 추가 가능)
                    - onClick: 아직 구현하지 않고, 스텁 상태 유지

핵심요구사항) 기존의 CSS와 스타일을 일관되게 유지할 것.
                - 색상: global.css의 변수 토큰 활용
                - 타이포그래피: 기존 스타일 준수
                - 아이콘 크기/여백: 일관성 있게 적용
