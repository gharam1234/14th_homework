핵심요구사항) Zustand 기반 필터 상태 관리 훅을 다음 조건으로 구현할 것.

1) Zustand Store 정의 (src/store/filterStore.ts)
   1-1) 상태 구조
        - availableNow: boolean (기본값: true - 즉시 구매 가능)
        - dateRange: { startDate: string | null, endDate: string | null } (기본값: { startDate: null, endDate: null })
        - keyword: string (기본값: "")

   1-2) 액션 함수
        - setAvailableNow: (value: boolean) => void
        - setDateRange: (startDate: string | null, endDate: string | null) => void
        - setKeyword: (value: string) => void
        - resetFilters: () => void (모든 필터 초기화)

   1-3) 로컬스토리지 연동
        - 상태 변경 시 자동으로 localStorage에 저장할 것.
        - 키명: "phone-filters"
        - 페이지 새로고침 시 로컬스토리지에서 상태 복원할 것.

2) 커스텀 훅 구현 (src/components/phones-list/hooks/index.filter.hook.ts)
   2-1) usePhoneFilters 훅 생성
        - Zustand 스토어와 연동하여 필터 상태 반환
        - 반환값:
            - availableNow: boolean
            - dateRange: { startDate: string | null, endDate: string | null }
            - keyword: string
            - setAvailableNow: (value: boolean) => void
            - setDateRange: (startDate: string | null, endDate: string | null) => void
            - setKeyword: (value: string) => void
            - resetFilters: () => void
            - isSearchEnabled: boolean (키워드 2자 이상이면 true)

   2-2) 유효성 검증 로직
        - isSearchEnabled: keyword.trim().length >= 2
        - 특수문자 허용 (제한 없음)

3) 컴포넌트 연동
   3-1) src/components/phones-list/index.tsx 수정
        - usePhoneFilters 훅 import 및 사용
        - 토글 버튼에 availableNow 상태 연결 (data-testid="toggle-available")
        - 날짜 선택기에 dateRange 상태 연결 (data-testid="datepicker")
        - 검색 입력창에 keyword 상태 연결 (data-testid="search-input")
        - 검색 버튼 활성화/비활성화를 isSearchEnabled로 제어 (data-testid="search-button")
        - 초기화 버튼 추가 및 resetFilters 연결 (data-testid="reset-button")
