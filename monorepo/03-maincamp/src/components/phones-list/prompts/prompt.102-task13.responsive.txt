## 핵심요구사항

1) 전역 컨테이너 레이아웃
   1-1) 최대 너비: 1280px (데스크톱)
   1-2) 패딩: 20px (모바일), 24px (태블릿), 40px (데스크톱)
   1-3) 마진: 자동 (가운데 정렬)
   1-4) 갭: 64px (섹션 간 수직 갭)

2) 캐러셀 영역 반응형
   2-1) 모바일: 너비 100%, 높이 자동
   2-2) 태블릿: 너비 100%, 높이 조정
   2-3) 데스크톱: 너비 100%, 높이 628px
   2-4) 화살표 버튼 크기: 모바일 30px → 데스크톱 40px

3) 프로모션 배너 반응형
   3-1) 모바일: 전체 너비
   3-2) 태블릿: 전체 너비
   3-3) 데스크톱: 전체 너비 (1280px)

4) 탭 & 검색/필터 영역 반응형
   4-1) 모바일
        - 탭: 스택 레이아웃 또는 수평 스크롤
        - 검색: 입력 필드 너비 100%, 버튼 아래로
        - 브랜드 필터: 수평 스크롤 (좌우 스크롤바)
   4-2) 태블릿
        - 탭: 나란히 배치
        - 검색: 입력 + 버튼 한 줄
        - 브랜드 필터: 수평 스크롤
   4-3) 데스크톱
        - 탭: 나란히 배치
        - 검색: 입력 + 버튼 한 줄 (gap 16px)
        - 브랜드 필터: 수평 스크롤 또는 한 줄

5) 카드 그리드 반응형
   5-1) 모바일(< 375px)
        - 1열
        - 카드 너비: 100% - padding
        - 갭: 16px
   5-2) 모바일L(376-667px)
        - 2열
        - gap: 16px
   5-3) 태블릿(668-1024px)
        - 3열
        - gap: 24px
   5-4) 데스크톱(> 1024px)
        - 4열
        - gap: 32px

6) 타이포그래피 반응형
   6-1) 제목: 28px (모바일) → 36px (데스크톱)
   6-2) 카드 모델명: 14px (모바일) → 16px (데스크톱)
   6-3) 카드 가격: 14px (모바일) → 16px (데스크톱)

7) 터치 제스처 지원
   7-1) 모바일: 스와이프로 캐러셀/브랜드 필터 조작
   7-2) 캐러셀: touch-action: pan-y (좌우 스와이프만 가능)
   7-3) 탭/카드: 클릭/탭 가능한 영역 최소 44px x 44px 보장

8) 레이아웃 최적화
   8-1) 모바일: flex 기반 (grid는 복잡성 증가)
   8-2) 데스크톱: grid 사용 (성능 최적)
   8-3) 스크롤 최적화: overflow-x auto (수평 스크롤 요소)
   8-4) 숨김 요소: display: none (미디어 쿼리)

9) CSS 구조화
   9-1) CSS Modules 사용
   9-2) 변수: --mobile, --tablet, --desktop 등으로 값 재사용
   9-3) mixin 또는 유틸 함수로 미디어 쿼리 반복 제거
   9-4) 주석: 각 섹션 구분 명확하게
